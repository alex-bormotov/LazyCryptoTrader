# LazyCryptoTrader

This is my first project :)

Install:

pip3 install mpu

pip3 install ccxt

pip3 pyTelegramBotAPI

[Create telegram bot](https://core.telegram.org/bots#3-how-do-i-create-a-bot)

Put your telegram bot api key in telegram_api.conf 

Start:

python3 LazyCryptoTrader.py
